# Erwin_Azucena_Project_2: The Handfuls of Community

# Project Overview

The project that I picked is a digital exploration of shared vulnerability that is inspired by Hua Xi's poem Handfuls. It reimagines "community" as not just a physical neighborhood that you see everyday, bvut as a collective emotional atmosphere, an "inner weather" that we all carry.

![Snow in Summer](<img/Snowing on a Summer Day.png>)

My goal was to move away from the rigid tradirional definition of community and to instead highlight the "collective emotional labor" that me and the people of the community perform. And even in moments of isolation like the pandemic, we are linked by the shared experience and memories of navigating sadness and resilience that inflict upon us. I have chosen Xi's imagery of snow falling on a summer day to repsent how our internal struggles manifest in the outside world beyond our community. This project serves as a tribute to the "ordinary people" who have look out for another durning blackouts and natural disaster that prove to me that empathy connects us even when we're not physically around.

# Technical Overview

The visual experience shown is built using a combaination of CSS Keyframe Animations and Photoshop Gifs to simulate the "lightly billowing effect described in the poem itself.

    *Animations: I utilized the opacity fades and translate Y to create a sense of falling snow that make you feels both weiighted and ethereal.
   
    * Transformations: Using scale and rotate properties on individual "tuft" element, that I ensured that no two visual elements move identically, mirroring the unique "handfuls" each person carries.
   
    *Tech Stark: The project is written in HTML5 and CSS3, developed within Visual Studio Code.

    *Hosting: The live environment is deployed via the GitHub Pages, in a way that allow a a seamless, high-performance web experience.
